# MaU
Android团队项目
## 运行环境
* JDK 1.7
* compileSdkVersion 25
* minSdkVersion 19
* targetSdkVersion 22
## 项目风格
* 高仿了新浪lofter的Android端
* 界面美观
* 多种动画
* 较多比较困难的控件实现
